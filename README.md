# 🚀 Optimization Developer Dashboard

A pro-level performance optimization dashboard that demonstrates how optimized code improves execution speed using real-time benchmarking and visualization.

---

## 🔥 Features

* ⚡ Compare slow vs optimized algorithms
* 📊 Live updating performance charts
* 🔄 Auto-running benchmark mode
* 📏 Multiple test sizes (Small / Medium / Large)
* 🎮 FPS-style performance meter
* ⏳ Loader animation during execution
* 📁 Export benchmark results as CSV

---

## 💡 Use Case

This project helps developers:

* Measure execution time in milliseconds
* Visualize performance improvements
* Understand algorithm efficiency (O(n²) vs O(n))
* Test performance under different workloads
* Track consistency with repeated runs

---

## 🛠️ Technologies Used

* HTML
* CSS (Modern UI + Glassmorphism)
* JavaScript
* Chart.js (for data visualization)

---

## ▶️ How to Run

1. Download or clone the repository
2. Open the folder in VS Code
3. Open `index.html`
4. Run using Live Server (recommended)

---

## 📊 How It Works

* The dashboard runs two functions:

  * Slow (O(n²))
  * Fast (O(n))
* It measures execution time using `performance.now()`
* Displays results and updates graph in real-time
